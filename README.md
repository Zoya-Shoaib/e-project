# Rainwater Harvesting Website

## Overview
This project is a responsive website developed to spread awareness about rainwater harvesting and its importance in water conservation. It presents information using a clean, simple layout that works smoothly across all devices.

## Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  
- jQuery  
- Font Awesome 4.7  

## Features
- Fully responsive design for mobile, tablet, and desktop
- Clean and beginner-friendly user interface
- Structured layout using Bootstrap grid system
- Use of Font Awesome icons for visual clarity
- Simple and readable front-end code

## Purpose
The purpose of this project is to demonstrate the implementation of responsive web design using HTML, CSS, Bootstrap, JavaScript, and jQuery. It applies basic front-end development concepts to present structured content in a clear and user-friendly manner using a real-world use case.
